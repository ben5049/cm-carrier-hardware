# Compute Module 5 Carrier Board (WIP)

Compact CM5 carrier board with an M.2 E-key connector and a 1000BASE-T1 ethernet PHY, as well as other optional IO.

- V1 Uses a separate PCIe switch and PCIe-to-ethernet bridge. It has been abandoned in favour of V2
- V2 Uses the PCI11010C which integrates a PCIe switch and ethernet MAC, leaving to space and power savings compared to V1
